# 🎮 Tic-Tac-Toe

A simple terminal-based Tic Tac Toe game written in C++, designed for two players with a clear and styled ASCII grid.

## 🖼️ Preview

<pre>
.-----------.
| X | O | X |
:---+---+---:
| O | X | O |
:---+---+---:
| X | O | X |
'-----------'
</pre>

## ⚙️ Requirements

- A C++ compiler (e.g., `g++`, `clang++`)

## 🚀 Getting Started

To compile and run:

```bash
g++ -o tic-tac-toe main.cpp
./tic-tac-toe
```
